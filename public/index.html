URL Shortener

FreeCodeCamp project built using Node Express and mongodb.

Built by Marc Kimbrey

https://github.com/marckimbrey

https://www.linkedin.com/marc-kimbrey

To get shortened URL /new/ + URL to shorten

example:
https://little-url.herokuapp.com/new/https://www.google.com

To vist shortened URL / + shortened URL

example
https://little-url.herokuapp.com/HyCgAb
